# React + TypeScript + Vite

This project is built using the following technologies:

- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.
- **Vite**: A build tool that provides fast and optimized development experience for modern web projects.
- **@tailwindcss/vite**: A plugin that integrates Tailwind CSS with Vite.
- **@tanstack/react-router**: A library that provides React Router integration for @tanstack/router.
- **@xyflow/react**: A library that provides React components and hooks for building flow-based UIs.
- **class-variance-authority**: A utility for generating safe and consistent CSS class names.
- **clsx**: A utility for composing CSS class names.
- **lucide-react**: A library of React icons.
- **next-themes**: A library that provides a simple way to manage and switch themes in Next.js.
- **radix-ui**: A set of primitives for building high-quality UIs.
- **zustand**: A tiny state management library.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the development server using `npm run dev`.
4. Open your browser and navigate to `http://localhost:3000`.
